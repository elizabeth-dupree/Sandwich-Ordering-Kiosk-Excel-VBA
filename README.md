# Sandwich-Ordering-Kiosk-Excel-VBA
Built a user interface and backend processing so that customers at the Cheese Shop in Williamsburg can order on a kiosk and employees are not overwhelmed with long lines

## Overview:
This project is an automated sandwich ordering kiosk built using Excel VBA. It simulates the ordering process at the Cheese Shop in Williamsburg, Virginia. The kiosk was designed to speed up the ordering process as the Cheese Shop still takes all orders by hand, which can cause long wait times for customers.

## Features
- Interactive Menu and Ingredient Selection
- Automated Price Calculation
- Order Summary Output
- Ingredient Inventory Report
- Button Based Navigation

## Skills Demonstrated
- Loops and Conditional Statements
- Subroutines and Functions
- Logical Workflow Development
- User Interface Design in Excel

## How to Run
- Download the .xlsm file from /app
- Open in Excel
- Enable Macros
- Click "Order Now"
